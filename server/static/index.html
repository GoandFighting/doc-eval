场景一：测试我的提取工具

  1. 选择内置数据集 newbench，点击「下载 PDF 数据集」
  2. 用您的工具将 PDF 转为 Markdown（文件名保持一致）
  3. 上传所有 .md 文件，点击「开始评测」
  4. 查看总分、各维度得分，下载报告

场景二：使用自己的数据集

  1. 点击「+ 上传自定义数据集」，上传含 .jsonl 规则的 ZIP 包
  2. 在下拉框选择您的数据集
  3. 用工具将 PDF 转为 Markdown，上传 .md 文件
  4. 点击「开始评测」，查看结果
